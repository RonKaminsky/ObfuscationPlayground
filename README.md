# ObfuscationPlayground

[![js-semistandard-style](https://img.shields.io/badge/code%20style-semistandard-brightgreen.svg?style=flat-square)](https://github.com/standard/semistandard)

An easily auditable web application for generating cryptographic hashes and obfuscating small amounts of data.

Tip: the parameter value can be in any format which Javascript's parseInt() understands, so binary values (for example, generated by hash functions) can be input in hexadecimal by prepending "0x". Parameter values are limited to 32 bits because of current WebAssembly limitations.
